Weather app

The layout for this weather app was inspired by a japanese poster from the 1990's. This weather app was designed using HTML, CSS and Javascript. The API that was linked was sourced from https://openweathermap.org/api. Users are able to search for the weather in any city and convert the temperature from celcius to fahrenheit. The lines under the city box is morse code.
